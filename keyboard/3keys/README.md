3 keys keyboard firmware
======================

3 keys fork : volume up/down and mute

## Build

Move to this directory then just run `make` like:

    $ make -f Makefile

## Keymap

Row is connected to F0
Cols are connected to F7  B6  B5 

### Layout
    ,--------------.
    |vold|mute|volu|
    `--------------'
